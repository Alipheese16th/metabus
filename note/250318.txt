# pdf week07
인포그래픽 (정보와 그래픽의 합성어)
복잡한 데이터와 정보를 시각적으로 표현
다양한 그래프를 활용하여 인포그래픽을 디자인해보자

꼭짓점 선택해서 Rotate 돌리기 꽃잎 5장 만들기
꽃잎을 Object - Graph - Design 으로 새 디자인 생성
그래프 우클릭 - Marker - 꽆잎 디자인에 매핑

각종 그래프를 사용해보면서 그림들을 디자인으로서 적용 시켜보기

그림에 선을 하나 기준선으로 그리고
view - guides - make guides 해서 가이드라인으로 적용시키고
선과 그림을 한번에 선택해서 object - graph - design 으로 적용시키면
다른 그래프에 사용할때 가이드라인 선 기준으로 그림이 늘어나게 된다.

# !! 넥서스폰트
설치하지않아도 사용할수있는 폰트 라이브러리

넥서스폰트 설치 후 샘플그룹에 내가 지정한 폴더와 연결 해놓으면
그 폴더에 모든 폰트 다 갖다 박아놓으면 된다.
굳이 폰트 설치 하지 않아도 적용 가능


텍스트 선택하고 Type - Create Outlines 을 사용하면 텍스트가 도형 타입으로 바뀐다
그럼 다른 컴퓨터에서 열어도 해당 서체가 없음에도 확인 가능하다.

글씨가 많아서 오타검수때문에 다른사람에게 보내는경우에는 텍스트형식으로 폰트까지 보내는게 맞다
