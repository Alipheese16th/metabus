# 게임 디자인 기초작업
- 콘셉트 : 장르와 플레이 스타일을 포함한 큰 밑그림과 게임 디자인
- 코어 메카닉 : 캐릭터가 할수 있는 플레이 기능이나 상호작용. 점프 슈팅 퍼즐 드라이빙
- 제어 방식 : 캐릭터 환경 상호작용 기타 실행 가능한 동작을 제어하는 버튼 맵 키 맵
- 스토리 : 게임 이야기 구조
- 아트 스타일 : 게임 전반에 걸친 일관된 스타일과 느낌
- 승패 조건 : 일반적으로 목표가 실패하도록 구성

# 게임 디자인 문서
- 게임디자인문서 : 게임 플레이 방법, 분위기, 스토리, 창작 경험
- 기술디자인문서 : 하드웨어부터 클래스 및 프로그램 아키텍처
- 원페이지 : 마케팅이나 프로모션에 사용, 게임의 스냅샷
