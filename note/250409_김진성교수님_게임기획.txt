

# 재미란

패턴: 새로운 규칙이나 패턴을 학습할때 느끼는 쾌감.

도전과 성취 : 목표를 정하고 거기에 달성하는 성취감

보상 시스템 : 특정행동을 했을때 받는 긍정적 피드백


----

기획은 크게 두가지

1. 콘텐츠 기획
- 시나리오 (스토리, 세계관)
- 몬스터
- npc (퀘스트)

2. 시스템 기획
- 전투
- 성장 (레벨)
- 인터페이스



----

원시트 기획서

게임 제목: 
장르: (예: 로그라이크 액션 어드벤처)
시놉시스: (게임의 배경, 세계관, 주요 목표와 전개 방식 등을 포함한 간략한 줄거리)
핵심 콘셉트: (무엇이 이 게임을 독특하게 만드는가?)
핵심 루프: (예: 전투 → 보상 → 업그레이드 → 다음 층)
도전 요소: (적, 함정, 시간 제한 등)
보상 시스템: (아이템, 스킬 강화, 해금 요소 등)
플레이어의 선택: (무기 선택, 경로 분기, 캐릭터 성장 경로 등)
플랫폼 / 플레이 방식: (PC, 모바일 / 싱글 or 멀티)



----


버드런

각종 장애물을 피해서 시작지점부터 목표지점까지 도착

레벨 별 맵 디자인 구성 달라짐.

bgm 유쾌하고 활기차고 엉뚱함

중간중간 세이브포인트 있음. (세이브포인트 디자인 어케할지 흠.. 작은새가 날라가는게 좋을듯)


스토리
독사가 알을 노렸으나
그순간 알에서 부화해서 독사를 따돌리고
계속해서 도망치는 스토리.
그러나 트리거일뿐 독사는 나오지않음. 그냥 맵 진행하기 위한 스토리
아닌가? 최종보스로 나와도 괜찮을듯


주인공
주인공은 잉꼬 얼굴 우스꽝스럽게 생김 (색상 하얀색 바탕의 노란색 하이라이트)
움직이기 wasd
점프키 스페이스 (점프할때 마다 새소리남)
알던지기 마우스 클릭 
구르기 ctrl (0.1초 무적 줄까)
아이템 사용


세이브포인트 (난이도 상승시 일회용으로 변함, 더 난이도 상승시 세이브 없음)


보스 (난이도 상승시 보스 체력 상승)
독수리 왜가리 항금알을 낳는 거위 등
특정 포인트를 알 던져서 맞춰야 피가 까임
보스종류마다의 패턴으로 공격 (거위는 황금알을 던짐)


스킬
보스 잡으면 해당 보스 스킬을 얻을수 있음 (패시브)
독수리 : 점프를 꾹누르면 활공
거위 : 그냥 알 대신 황금알 던지기 (투사체가 더 크고 폭발범위가 큼, 데미지 상승)


아이템
맵 진행시 중간중간 새장이 있음 새장을 공격해서 맞추면 아이템 획득
(이미 갖고있는 아이템이 있으면 안먹어지고 해당 자리에 카드로 떨어져있음)
1. 일회성 세이브를 바로 생성
2. 잠깐동안 날수있음
3. 사용시 점프 (공중에서 사용가능하기때문에 2단점프로 활용. 기본 점프와 별개이므로 먼저 사용후 기본점프도 가능)
4. 잠시 스피드가 빨라지고 무적상태


장애물
이동 방해하는 장애물이 주컨텐츠
맵 자체도 난해해서 점프해서 올라가기 어렵게 만듬


잔몹
보통 체력1이나 좀 튼튼한애는 체력2라서 일반공격은 두대, 황금알로 던져야 한방


맵
숲, 사막, 설산, 화산 정도 컨셉으로 
갈수록 장애물이나 몬스터 난이도 어려워짐
그러나 오아시스같은 휴식처가 한두개 있는게 좋을듯

# 애프터이펙트

##
상단의 툴 중에
Puppet position pin tool 이 있다.
이걸 움직일 위치에 (뼈대심기) 적용해놓고

키프레임마다 위치를 조졍해가면
마치 그림이 살아움직이는것처럼 표현됨.

그리고 반복시키려면 키프레임을 복사하기엔 노가다니까

Puppet Pin 속성중에 시계아이콘을 ALT 누르면서 클릭

Property - loopOut 옵션을 주면 무한반복함


이런식으로 퍼펫툴을 한번 살펴봤다.
그림에 움직임을 주는 툴.


## 외계인 그림 예제
다른 파일 불러올때 Layer 로 불러와야 이미지 하나씩 조정 가능
팔을 선으로 따서 그리고 키프레임마다 움직임 주기.



## 자전거 예제

### 자전거 바퀴만 키프레임별 로테이션 돌려서 바퀴 구르게 움직임

### 페달은 제일 중심이 되는 기어에다 나머지 부품을 parent 로 걸고 
중심축 기어를 회전시키면 같이 돔.
밟는부분은 반대로 회전시키면 가만히 있는것처럼 됨

### 다리
다리를 mask 사용해서 4가지 부위로 자른다.

Pan Behind 앵커툴 을 사용해서 부위별 앵커를 관절 위치에 두고

각 부위별 parent를 연결한다. 발 -> 다리 -> 허벅지 -> 골반

키프레임 마다 페달위치에 맞춰서 각부위를 회전시킨다
